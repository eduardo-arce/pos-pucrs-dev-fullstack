# 🚨Arquitetura-Client-Side🚨

# Sobre a Disciplina

Introdução à arquitetura client-side, com atenção às camadas front-end (HTML, CSS, JS), back-end e intermediária. A disciplina discute convergências como mobilidade, nuvem e IoT, padrões arquiteturais (três camadas, clean, hexagonal) e os elementos necessários para uma comunicação eficiente entre cliente e servidor.

# Conceitos chave

- Arquitetura client-side: organização em front-end, back-end e camada intermediária, pensada para sistemas distribuídos  
- Business agility: capacidade de responder rapidamente a mudanças do mercado e às novas demandas dos clientes  
- Fundamentos web: URL/URI/URN para identificar recursos, CDN para entregar conteúdo e protocolos HTTP para requisições  
- Padrões arquiteturais: três camadas (comunicação entre elas), onion (estrutura em camadas) e hexagonal (adaptadores de entrada/saída)  
- Elementos essenciais: dispositivos, plataformas (browsers e OS), mensageria (chatbots) e design system para consistência visual  
- Exemplos práticos: componentes em React, roteamento com Next.js, integração de APIs e uso de templates já existentes  

# Ferramentas iniciais

HTML/CSS/JavaScript, React, Next.js, CDNs, meta tags e design systems
