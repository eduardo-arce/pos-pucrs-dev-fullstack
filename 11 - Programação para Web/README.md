# 🚨Programação-Web🚨

# Sobre a Disciplina

Ponto de partida no desenvolvimento de aplicações web, com ênfase em HTML, CSS e JavaScript. A disciplina se dedica à manipulação do DOM (Document Object Model), ao uso de formulários, à construção de aplicações responsivas e acessíveis e ao entendimento da evolução da web, suas arquiteturas e protocolos.

# Conceitos chave

- Evolução da web: da 1.0 (conteúdo estático) até a 3.0 (inteligente, com IA, segurança e criptografia)  
- Arquiteturas web: cliente-servidor, orientada a serviços (SOA) e microsserviços (independentes, com APIs e flexibilidade)  
- Protocolos: TCP (conexão confiável) versus UDP (rápido, sem garantias), além de HTTP/HTTPS, FTP e DNS  
- Dimensões de projeto: estrutural (metas/objetivos), navegacional (como se navega) e de apresentação (visual)  
- Características de apps web: acesso via navegador, dados na nuvem, interatividade, atualizações automáticas e segurança  
- Boas práticas: pensar em escalabilidade, flexibilidade, desempenho, padrões e segurança desde o design  

# Ferramentas iniciais

HTML/CSS/JS, manipulação do DOM, formulários, recursos para design responsivo (media queries) e navegadores para testes
