# 🚨Arquitetura-de-Software-na-Nuvem🚨

# Sobre a Disciplina

Introdução à arquitetura de software em ambientes de nuvem, com destaque para datacenters distribuídos, provedores de cloud, containers, eficiência de performance, estratégias de backup na nuvem e sustentabilidade em TI.

# Conceitos chave

- Nuvem e datacenters: a nuvem se apoia em estruturas de datacenters distribuídos, sem abrir mão deles  
- Provedor de nuvem: entidade responsável por manter serviços disponíveis para as partes interessadas  
- Container: virtualização no nível do sistema operacional, ideal para implantar e executar aplicações distribuídas  
- AWS: plataforma com a maior variedade de tipos de máquinas virtuais para computação  
- Eficiência de performance: uso racional de recursos, adaptando-se à demanda e à evolução tecnológica  
- Backup na nuvem: mesmo em ambientes cloud continua sendo uma peça indispensável  
- Acesso remoto: computação em nuvem via internet, com armazenamento de dados relevante e atenção à sustentabilidade

# Ferramentas iniciais

AWS (máquinas virtuais, serviços), containers para virtualização e ferramentas de backup e monitoramento de performance
